View Project

Project Overview

Este proyecto analiza el desempeño de ventas mediante la integración de datos transaccionales desde SQL, permitiendo evaluar ingresos, volumen de ventas, ticket promedio y participación por categoría de producto.

Se desarrollaron medidas DAX para calcular crecimiento mensual, ranking de productos y análisis de clientes clave. El dashboard incluye filtros dinámicos por región, canal de ventas y periodo.

El objetivo fue identificar patrones de compra, detectar productos con bajo rendimiento y apoyar estrategias comerciales basadas en datos.
