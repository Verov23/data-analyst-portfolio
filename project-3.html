Customer Segmentation with Python

Segmentación de clientes utilizando técnicas de clustering (K-Means) para identificar grupos de comportamiento de compra y apoyar estrategias de marketing basadas en datos.

Ver Proyecto

Project Overview

Este proyecto aplica técnicas de análisis exploratorio y clustering para segmentar clientes según variables como frecuencia de compra, gasto total y valor promedio de pedido.

Se realizó limpieza y transformación de datos con Pandas, normalización de variables y aplicación del algoritmo K-Means. Se utilizó el método del codo para determinar el número óptimo de clusters.

Los resultados permitieron identificar segmentos de clientes de alto valor, clientes en riesgo y oportunidades de campañas de retención.
