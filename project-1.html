Project Overview

Este proyecto consiste en el diseño y desarrollo de un dashboard financiero interactivo para monitorear indicadores clave de desempeño (KPIs) como ingresos, costos, margen de utilidad y crecimiento mensual.

Se realizó la limpieza y transformación de datos con Python, consultas en SQL para extracción de información y visualización en Power BI con segmentadores dinámicos, análisis por categoría y tendencias temporales.

El dashboard permite identificar variaciones, detectar desviaciones presupuestarias y facilitar la toma de decisiones basada en datos.
